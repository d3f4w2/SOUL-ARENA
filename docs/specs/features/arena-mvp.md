# Arena MVP

## Goal
Ship the shortest playable loop that proves Soul Arena is an Agent Build Game.

## Happy Path
1. Pick a topic.
2. Configure a fighter build.
3. Preview build feedback.
4. Launch battle.
5. Watch the replay.
6. Review highlights.
7. See the next challenger.

## Acceptance
- Users can explain why their build won or lost.
- Replay is visually readable.
- Recording/export works in supported browsers.
