# SecondMe Agent Ingestion

## Goal
Use current identity integration to seed the player-facing fighter identity.

## MVP Scope
- read current user profile if logged in
- optionally seed fighter name and tags from SecondMe
- do not block local mock play when not logged in

## Acceptance
- Logged-in users can start from their own name/profile context.
- Logged-out users can still use the arena with manual input.
